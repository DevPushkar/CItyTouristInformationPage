Please follow this step to run this project:
(1) Copy the clone URL and run the git clone command in any empty folder Ex: (git clone CLONE_URL)
(2) Install npm packages/dependencies (npm install)
(3) Run the project by using
    - npm start
    - nodemon (Auto restart the server when any file changes
(4) Go to your web browser and navigate to localhost:3000
(5) Enjoy!
